# Gallery
An android app that displays the user's gallery from an API and the favorite photos from a local storage.
<hr>

<h2>Permissions</h2>
- Network Permission.
<hr>

<h2>Technologies Used:</h2>
- Android Studio.
<br>
- Kotlin.
<br>
- Retrofit.
<br>
- DiffUtil.
<br>
- Flow with coroutine.
<br>
- Dependency Injection.
<br>
- Room.
<br>
- Navigation component.
<br>
- MVVM.
<br>
- Clean Architecture.
